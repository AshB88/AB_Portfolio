# AB_Portfolio

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Description
The AB_Portfolio project is a professional web developer portfolio built using React. This single-page application showcases my skills, projects, and experience in web development. The portfolio is designed to provide potential employers and collaborators with a comprehensive overview of my work and capabilities.

### Features

- **Responsive Design**: The portfolio is built with a mobile-first approach, ensuring a seamless experience across all devices.
- **Modern Technologies**: Utilizes React for efficient component-based development and Vite for fast build times and optimized performance.
- **Navigation**: Includes a header with navigation links to different sections of the portfolio, such as About Me, Portfolio, Contact, and Resume.
- **Project Showcase**: Displays titled images of six of my applications, each with links to the deployed applications and their corresponding GitHub repositories.
- **Contact Form**: Features a contact form with fields for name, email address, and message, including validation to ensure required fields are filled and email addresses are valid.
- **Downloadable Resume**: Provides a link to download my resume in PDF format and lists my proficiencies.
- **Footer**: Contains links to my GitHub, LinkedIn, and Stack Overflow profiles.

## Installation
1. **Clone the repository**:
    ```sh
    git clone https://github.com/AshB88/AB_Portfolio.git
    ```
2. **Navigate to the project directory**:
    ```sh
    cd AB_Portfolio
    ```
3. **Install the dependencies**:
    ```sh
    npm install
    ```
4. **Start the development server**:
    ```sh
    npm run dev
    ```
5. **Open your browser and navigate to the prompted URL to view the application**.

## Usage
To use this portfolio, simply navigate through the different sections using the navigation links in the header. You can view my projects and download my resume.
